# Portfolio Website — HTML + CSS  
Итоговая практическая работа  

---

##  О проекте

Это адаптивный персональный сайт-портфолио, созданный для демонстрации навыков HTML и CSS. 
 
Проект соответствует требованиям учебной практической работы:
- использованием семантической структуры HTML5  
- построением сеток на CSS Grid  
- использованием Flexbox  
- адаптивной версткой (mobile-first)  
- применением современных CSS-селекторов (`:has()`)  
- использованием медиа-запросов  
- добавлением мультимедиа (видео и аудио)  
- созданием боковой колонки `<aside>`  
- реализацией адаптивного меню-бургера **без JavaScript**

Сайт включает такие разделы, как:
- **Обо мне**
- **Презентация**
- **Портфолио**
- **Мои сильные стороны**
- **Контакты**
- **Дополнительный aside**
